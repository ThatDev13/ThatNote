ThatNote

Markdown / LaTeX Studio

Start crafting a clean note.

Import a file or begin from a blank canvas. Switch between Markdown and LaTeX in the editor.

Tips:

  • Use `$E=mc^2$` for inline math, `$$\\int_0^1 x^2 dx$$` for blocks.
  • Export saves notes as .md files.
  • Use the PDF button to download a PDF version of your note.
  • The New Session button allows you to start a fresh note.
  • The Reset button clears the current note and starts a new session.
  • The Export button saves your note in the selected format.

# ThatNote

## Beta Editor Mode

Der Editor enthält einen neuen **Beta Mode** mit Glass-Design und integriertem KI-Assistant.

### Features

- Beta-Mode-Toggle direkt im Editor
- Fullscreen-Split im Beta-Mode (links Editor, rechts Preview)
- Kein vertikales Seitenscrolling im Beta-Mode
- Kreisförmiger AI-FAB unten rechts
- AI-Chat öffnet sich unten rechts
- AI schreibt direkt in den Text, danach Accept/Reject möglich
- Live Statistik: Wörter und Zeichen

### OpenAI Key

Der Editor nutzt einen fest hinterlegten Default OpenAI API-Key im Frontend-Code.
